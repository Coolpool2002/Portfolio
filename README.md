Hayden Jones – Hardware, Systems & Software Portfolio

Welcome to my project portfolio. I'm a hands-on tech enthusiast with a passion for designing, modifying, and building systems from the ground up. My work spans electronics diagnostics, embedded systems, software development, 3D design, and practical repair skills across a wide range of tools and platforms.

A large part of my workflow involves 3D printing and rapid prototyping. I use an Ender 3 V3 SE printer to bring designs to life, producing strong, functional parts in materials such as PLA, TPU, PETG, and ABS. From structural brackets to flexible mounts, I tune prints for strength, precision, and durability depending on the use case.

I regularly work with embedded systems like Arduino and Raspberry Pi, integrating them with custom circuitry, sensors, and user interfaces to build smart devices, mods, and automated tools. This includes soldering, breadboarding, and crafting enclosures in CAD software such as FreeCAD and Fusion 360.

I also develop interactive software and games in JavaScript and Python. A recent highlight includes a fully browser-based Flappy Ball leaderboard system with user login, real-time Firebase integration, and game state logic in HTML5 Canvas. I build applications with a focus on clean code, fault tolerance, and full-stack design thinking—from UI to cloud database.

This site showcases a collection of my technical builds, programming projects, and engineered solutions—each focused on learning, improving, and solving real-world problems through hands-on innovation.
